Project Details

InStock

Instock is a warehouse and inventory management system that was built in a week by a team of four developers using Agile workflows (Jira) and GitFlow. Every morning we would have a stand-up to mimic a work environment. I really enjoyed working with Jira, GitFlow, and Scrum methodologies. It allowed us to take a project that felt daunting and break it down into manageable pieces. Instock is a fully responsive full-stack web application.
