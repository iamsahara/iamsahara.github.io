# 🧑‍💻 Sahar's Portfolio

A minimalist single-page developer portfolio built with Vite, SCSS, and Vanilla JS — designed to showcase my best work, skills, and personality.

## 🚀 Overview

This is my latest personal portfolio. It features smooth scroll navigation, custom animations, and a strong visual identity. No navbar — just clean, bold sections and good vibes ✨

## 🔧 Tech Stack

- Vite
- HTML5
- SCSS (modular structure)
- Vanilla JavaScript
- Lottie for animations
- Responsive design (Flexbox)

## 📸 Live Demo

[🌐 View Portfolio](https://iamsahara.github.io)

## 📂 Features

- 🎯 Clean hero section with intro and resume
- 🧩 Animated skill loops and scroll interactions
- 🧠 Project cards with GitHub + Live links
- 🎨 Styled with SCSS and modern responsive layout
- 💌 Contact section with custom interaction
